Verdiepingsonderzoek

Onderzoeksvraag

Wat zijn Learn To Earn modellen binnen het kader van web 3.0?


Context

Om de vraag “Hoe wordt de gebruikerservaring van een Learn To Earn model toegepast op de bestaande leeromgeving van Handpicked Agencies?” te kunnen beantwoorden, moeten er eerst wat deelvragen worden beantwoord. Een van deze deelvragen is: “Wat zijn Learn To Earn modellen binnen het kader van web 3.0?” . In dit onderzoek wordt deze vraag beantwoord. Voor dit onderzoek zijn er drie verdiepingsonderzoeken gedaan, dit document zal de kennis gebruiken van de vorige onderzoeken “Web 3.0 NFT’s”, “NFT-onderwerp verdieping” en “Hoe maak je NFTs?” .


Methodes

Voor dit onderzoek zijn de volgende CMD-methodes gebruikt.


Literature Study

Voor dit onderzoek zijn er veel verschillende online artikelen, wetenschapsdocumenten, whitepapers, informatieve video’s en technische documentatie gelezen om verdiept te worden over het onderwerp van Web 3.0.


Trend Analyse

Voor dit onderzoek is er gekeken naar wie de doelgroep is van web 3.0 technologie en hoe ze reageren op nieuwe ontwikkelingen. Ook is er gekeken naar de technische nieuwste trends rondom web 3.0 en hoe dit in het programmeren ontwikkeld.


Doelgroep Interview

Voor dit onderzoek is ook de stakeholder (die NFT’s bezit) gevraagd waarom hij NFT’s bezit en waarom hij bepaalde “To Earn” programma’s gebruikt.


Resultaten

De conclusie van het onderzoek staat in het kopje hieronder. Lees het document hieronder voor het uitgebreide onderzoek en conclusie met de bijhorende bronnen.

Conclusie

Wat zijn Learn To Earn modellen binnen het kader van web 3.0? Een Learn To Earn model binnen het kader van web 3.0 is een model waar je als gebruiker crypto tokens krijgt als beloning voor het voltooien van cursussen en toetsen. Je leerproces en verdiende tokens worden opgeslagen op een NFT. Je kunt de crypto token vervolgens als cryptocurrency gebruiken op het internet.


Vervolg

Nu het bekend is wat Learn To Earn modellen zijn, kunnen er wat ideeën bedacht worden voor eigen Learn To Earn toepassingen. Maar om echt concepten te kunnen maken, moet eerst de vraag “Welke elementen hebben bestaande Learn To Earn platformen binnen hun applicaties?” worden beantwoord. Dan kunnen pas echt concepten en low fidelity prototypes gemaakt kunnen worden.
